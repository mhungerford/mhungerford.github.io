Pico-8 RGB Arcade

Built an RGB Panel-based arcade running Pico-8 games on a 128x128 RGB Matrix. Uses some neat tricks to render directly from Pico-8 to the RGB Matrix software for higher refresh rates. Uses the Pico Arcade launcher with a custom set of games to provide a multi-game arcade experience with games such as Dinkey Kong, Pic-Man, Frogger, Bubble Bobble, Brawl, HeMan.
